# ImageToASCII
Converts the Image file in the form of ASCII characters for displaying the images in basic programs like notepad
Converts the Image file in the form of ASCII characters for displaying the images in basic programs like notepad

To run you would need Python 2.7 and python Pillow library

then got to the same folder as the python file in terminal and write

python img.py "path of the image file you want to convert without the double qoutes I have written here"
